# 股票资金流向监控工具

## 功能介绍

- 📊 获取股票分时资金流向数据（分钟级）
- 📈 生成资金流向**动画**（GIF格式）
- 💾 保存历史数据到CSV文件
- ⏰ **自动定时生成**：午盘(11:30)和收盘(15:30)后自动生成
- 🔥 板块资金流向**折线图对比**（所有板块在同一张图上）
- 🔄 **多源数据获取**：新浪(akshare)为主力源，东财为备用，自动切换
- ✅ **多API交叉验证**：确保数据准确性

## 专业级可视化 (v2.0)

### Bloomberg/Wind终端风格

- **深色主题**：专业金融终端配色方案
- **辉光效果**：线条和标记点带辉光，增强视觉层次
- **渐变填充**：零线上下方渐变填充，直观显示正负流向
- **专业排版**：统计面板、信息面板、进度条一体化布局
- **高分辨率**：130 DPI输出，细节清晰

### 动画特性

- **平滑过渡**：三次缓动函数，动画流畅自然
- **实时标记**：当前值带辉光环和阴影，醒目易读
- **专业图例**：半透明背景，边框清晰
- **进度指示**：底部进度条显示动画进度

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

依赖列表：
- `requests` - HTTP请求
- `pandas` - 数据处理
- `numpy` - 数值计算
- `matplotlib` - 图表生成
- `Pillow` - GIF动画
- `akshare` - 新浪数据源（稳定，不易被封）

### 2. 配置监控股票

编辑 `config.py`，修改 `STOCK_LIST`：

```python
STOCK_LIST = {
    "600519": "贵州茅台",
    "000858": "五粮液",
    "300750": "宁德时代",
}
```

### 3. 运行

#### 方式一：定时自动运行（推荐）

```bash
python scheduled_report.py
```

程序会自动在以下时间生成报告：
- **11:30** - 午盘报告（上午场 09:30-11:30）
- **15:30** - 收盘报告（全天场 09:30-15:00）

#### 方式二：手动运行

```bash
# 生成上午场报告
python flow_animator.py noon

# 生成全天场报告
python flow_animator.py close

# 实时监控（交易时间内自动刷新）
python realtime_monitor.py

# 板块资金流向监控
python sector_flow_monitor.py
```

#### 方式三：测试动画（使用模拟数据）

```bash
python test_animation.py
python test_sector_animation.py
```

## 数据源策略

系统采用多源自动切换策略，确保数据稳定获取：

| 优先级 | 数据源 | 特点 | 状态 |
|--------|--------|------|------|
| 1 | 东方财富 | 分时K线精细（分钟级） | 易被限流 |
| 2 | 新浪(akshare) | 稳定不易被封 | **推荐** |

- 优先尝试东财分时数据（更精细）
- 东财不可用时自动切换到新浪源
- 新浪源使用模拟曲线（基于真实排名数据）

## 输出说明

### 动画文件（GIF）

保存在 `charts/` 目录：

| 文件名 | 说明 |
|--------|------|
| `{代码}_{名称}_{日期}_noon_flow.gif` | 个股上午场动画 |
| `{代码}_{名称}_{日期}_close_flow.gif` | 个股全天场动画 |
| `sector_lines_{日期}_noon_flow.gif` | 板块上午场折线图 |
| `sector_lines_{日期}_close_flow.gif` | 板块全天场折线图 |

### 数据文件（CSV）

保存在 `data/` 目录：

| 文件名 | 说明 |
|--------|------|
| `{代码}_{名称}_{日期}.csv` | 个股分钟级资金流向数据 |
| `sector_{类型}_{日期}.csv` | 板块资金流向数据 |

## 动画内容

### 个股动画（双子图）

1. **上图 - 分时资金净流入**
   - 红色曲线：主力净流入（带辉光效果）
   - 蓝色曲线：小单净流入
   - 绿色曲线：中单净流入
   - 渐变填充：零线上方红色，下方绿色

2. **下图 - 主力累计净流入**
   - 橙色曲线：累计走势（带辉光效果）
   - 渐变填充：正值区域橙色渐变

3. **右侧面板**
   - 关键指标：实时数值显示
   - 数据信息：股票信息、数据点、时间范围

### 板块动画（折线图对比）

- **所有板块在同一张图上显示**
- 每条折线代表一个板块的主力累计净流入
- 动画逐步绘制各板块的走势
- 可以直观对比各板块资金流向的强弱
- 右侧实时排名面板
- 支持上午场/全天场两种模式

## 注意事项

- 需要在 **交易时间** 运行才能获取实时数据
- 数据来源：东方财富 + 新浪（免费接口，无需登录）
- **API限制**：东财请求过于频繁会被临时封禁，系统会自动切换到新浪源
- 非交易时间运行会使用最近一次的数据
- 定时报告系统支持跨天自动重置，无需手动重启
