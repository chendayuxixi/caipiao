# 股票资金流向监控工具

## 功能介绍

- 📊 获取股票分时资金流向数据（分钟级）
- 📈 生成资金流向**动画**（GIF格式）
- 💾 保存历史数据到CSV文件
- ⏰ **自动定时生成**：午盘(11:30)和收盘(15:00)后自动生成
- 🔥 板块资金流向**折线图对比**（所有板块在同一张图上）

## 专业级可视化 (v2.0)

### Bloomberg/Wind终端风格

- **深色主题**：专业金融终端配色方案
- **辉光效果**：线条和标记点带辉光，增强视觉层次
- **渐变填充**：零线上下方渐变填充，直观显示正负流向
- **专业排版**：统计面板、信息面板、进度条一体化布局
- **高分辨率**：130 DPI输出，细节清晰

### 动画特性

- **平滑过渡**：三次缓动函数，动画流畅自然
- **实时标记**：当前值带辉光环和阴影，醒目易读
- **专业图例**：半透明背景，边框清晰
- **进度指示**：底部进度条显示动画进度

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置监控股票

编辑 `config.py`，修改 `STOCK_LIST`：

```python
STOCK_LIST = {
    "600519": "贵州茅台",
    "000858": "五粮液",
    "300750": "宁德时代",
}
```

### 3. 运行

#### 方式一：定时自动运行（推荐）

```bash
python scheduled_report.py
```

程序会自动在以下时间生成报告：
- **11:30** - 午盘报告
- **15:00** - 收盘报告

#### 方式二：手动运行

```bash
# 生成所有动画（个股 + 板块折线图）
python flow_animator.py

# 测试动画（使用模拟数据）
python test_animation.py

# 测试板块折线图动画（使用模拟数据）
python test_sector_animation.py
```

## 输出说明

### 动画文件（GIF）

保存在 `charts/` 目录：

| 文件名 | 说明 |
|--------|------|
| `{代码}_{名称}_{日期}_flow.gif` | 个股资金流向动画（双子图） |
| `sector_lines_{日期}_flow.gif` | **板块资金流向折线图对比** |

### 数据文件（CSV）

保存在 `data/` 目录：

| 文件名 | 说明 |
|--------|------|
| `{代码}_{名称}_{日期}.csv` | 个股分钟级资金流向数据 |
| `sector_{类型}_{日期}.csv` | 板块资金流向数据 |

## 动画内容

### 个股动画（双子图）

1. **上图 - 分时资金净流入**
   - 红色曲线：主力净流入（带辉光效果）
   - 蓝色曲线：小单净流入
   - 绿色曲线：中单净流入
   - 渐变填充：零线上方红色，下方绿色

2. **下图 - 主力累计净流入**
   - 橙色曲线：累计走势（带辉光效果）
   - 渐变填充：正值区域橙色渐变

3. **右侧面板**
   - 关键指标：实时数值显示
   - 数据信息：股票信息、数据点、时间范围

4. **底部进度条**
   - 动画进度指示
   - 当前时间显示

### 板块动画（折线图对比）

- **所有板块在同一张图上显示**
- 每条折线代表一个板块的主力累计净流入
- 动画逐步绘制各板块的走势
- 可以直观对比各板块资金流向的强弱
- 右侧实时排名面板

## 资金分类说明

| 类型 | 金额范围 |
|------|----------|
| 超大单 | >= 100万 |
| 大单 | 20万 - 100万 |
| 中单 | 4万 - 20万 |
| 小单 | < 4万 |

## 板块类型

| 类型 | 说明 | 示例 |
|------|------|------|
| industry | 行业板块 | 半导体、白酒、医药 |
| concept | 概念板块 | 人工智能、新能源 |
| region | 地域板块 | 北京、上海、广东 |

## 动画速度设置

| 速度 | 说明 | 适用场景 |
|------|------|----------|
| slow | 慢速（约8秒） | 详细观察 |
| normal | 正常（约4秒） | 日常使用 |
| fast | 快速（约1秒） | 快速浏览 |

## 注意事项

- 需要在 **交易时间** 运行才能获取实时数据
- 数据来源：东方财富（免费接口，无需登录）
- **API限制**：请求过于频繁会被临时封禁，请耐心等待
- 非交易时间运行会使用最近一次的数据
